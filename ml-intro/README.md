# 机器学习入门学习计划（参考西瓜书）

本目录是一份参考周志华《机器学习》（俗称"西瓜书"）的入门学习计划与配套案例代码，
面向已有 Python 基础的学习者。建议总学习周期约 **10 周**，每周 6~10 小时。

## 环境准备

```bash
pip install -r requirements.txt
```

依赖：numpy、pandas、scikit-learn、matplotlib。

每个案例都是独立脚本，直接运行即可，例如：

```bash
python code/ch03_linear_model/logistic_regression_watermelon.py
```

## 每周详细计划

### 第 1 周：绪论 + 模型评估与选择（第 1~2 章）
**学习目标**
- 理解基本术语：样本、特征、标记、假设空间、归纳偏好、过拟合/欠拟合。
- 掌握评估方法：留出法、交叉验证法、自助法。
- 掌握性能度量：错误率/精度、查准率 P、查全率 R、F1、ROC 与 AUC。

**动手实践**
- 运行 `holdout_cross_validation.py`：对比留出法与 10 折交叉验证的评估结果。
- 运行 `pr_roc_curves.py`：绘制 P-R 曲线与 ROC 曲线，计算 AUC。

**自测问题**
- 为什么不能用训练误差来评估泛化能力？
- 类别不平衡时，为什么精度（accuracy）可能失效？

### 第 2 周：线性模型（第 3 章）
**学习目标**
- 掌握线性回归的最小二乘法（闭式解）与梯度下降解法。
- 掌握对数几率回归（Logistic Regression）的极大似然推导与梯度更新。
- 了解线性判别分析（LDA）的思想。

**动手实践**
- 运行 `linear_regression.py`：分别用闭式解与梯度下降拟合线性回归，对比结果。
- 运行 `logistic_regression_watermelon.py`：在西瓜数据集 3.0α 上从零实现对数几率回归（书中习题 3.3）。

**自测问题**
- 对数几率回归为什么用对数似然而不是均方误差作为损失？

### 第 3 周：决策树（第 4 章）
**学习目标**
- 理解信息熵、信息增益（ID3）、增益率（C4.5）、基尼指数（CART）。
- 理解预剪枝与后剪枝。

**动手实践**
- 运行 `decision_tree_id3.py`：在西瓜数据集 2.0 上从零实现基于信息增益的决策树（书中习题 4.3 简化版），打印树结构。
- 运行 `decision_tree_sklearn.py`：用 scikit-learn 在鸢尾花数据集上训练并可视化决策树。

**自测问题**
- 信息增益为什么会偏好取值数目多的属性？增益率如何缓解？

### 第 4~5 周：神经网络（第 5 章）
**学习目标**
- 理解神经元模型（M-P 模型）、感知机及其局限（异或问题）。
- 掌握多层前馈网络与误差逆传播（BP）算法的推导。
- 了解全局最小与局部极小、常见激活函数。

**动手实践**
- 运行 `perceptron.py`：实现感知机，验证其能解线性可分问题但解不了异或。
- 运行 `bp_network.py`：从零实现标准 BP 算法（书中习题 5.5），在西瓜数据集 3.0 上训练单隐层网络解决异或和分类问题。

**自测问题**
- BP 算法中隐层梯度是如何由输出层梯度"逆传播"得到的？

### 第 6 周：支持向量机（第 6 章）
**学习目标**
- 理解间隔与支持向量、对偶问题、KKT 条件（概念层面）。
- 理解核函数（线性核、RBF 核）与软间隔。

**动手实践**
- 运行 `svm_kernels.py`：用 scikit-learn 对比线性核与 RBF 核在非线性数据上的决策边界（书中习题 6.2 思想）。

**自测问题**
- 为什么核技巧能让 SVM 处理非线性可分数据？

### 第 7 周：贝叶斯分类器（第 7 章）
**学习目标**
- 理解贝叶斯决策论、极大似然估计。
- 掌握朴素贝叶斯的"属性条件独立性假设"与拉普拉斯修正。

**动手实践**
- 运行 `naive_bayes_watermelon.py`：在西瓜数据集 3.0 上从零实现朴素贝叶斯（含拉普拉斯修正，对应书中例 7.1 / 习题 7.3），判断"测1"样本是否为好瓜。

**自测问题**
- 拉普拉斯修正解决了什么问题？

### 第 8 周：集成学习（第 8 章）
**学习目标**
- 理解 Boosting（AdaBoost）与 Bagging（随机森林）的区别。
- 理解"好而不同"：个体学习器的准确性与多样性。

**动手实践**
- 运行 `adaboost.py`：以决策树桩为基学习器从零实现 AdaBoost（书中习题 8.3 简化版）。
- 运行 `bagging_random_forest.py`：用 scikit-learn 对比单棵决策树、Bagging 与随机森林。

**自测问题**
- AdaBoost 每一轮如何调整样本权重？为什么？

### 第 9 周：聚类（第 9 章）
**学习目标**
- 理解聚类任务与性能度量（外部指标/内部指标）、常用距离。
- 掌握 k-means 算法流程。

**动手实践**
- 运行 `kmeans_watermelon.py`：在西瓜数据集 4.0 上从零实现 k-means（书中图 9.2 算法），可视化聚类过程。

**自测问题**
- k-means 的目标函数是什么？为什么它一定收敛？

### 第 10 周：降维与度量学习（第 10 章）
**学习目标**
- 理解 k 近邻学习与"维数灾难"。
- 掌握 PCA 的推导（最大可分性视角）与实现步骤。

**动手实践**
- 运行 `pca_knn.py`：从零实现 PCA 将鸢尾花数据降到 2 维并可视化，再用 kNN 在降维前后分别分类、对比精度。

**自测问题**
- PCA 中为什么选取协方差矩阵最大的 d' 个特征值对应的特征向量？

## 进阶方向（学完后）
- 西瓜书第 11~16 章：特征选择、计算学习理论、半监督、概率图模型、规则学习、强化学习。
- 《南瓜书》（datawhalechina/pumpkin-book）：西瓜书公式的详细推导。
- 深度学习：《动手学深度学习》（d2l.ai），配合 PyTorch 实践。
- 实战：Kaggle 入门竞赛（Titanic、House Prices）。

## 学习建议
1. **先看书再跑代码，再尝试自己复现**：每章先通读，再运行案例，最后合上书自己写一遍核心算法。
2. **公式推导不必一次吃透**：第一遍以理解思想为主，推导可借助南瓜书二刷。
3. **坚持做自测问题**：能向别人讲清楚，才算真正学会。
